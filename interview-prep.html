What this is
🗂
A portable, offline-first interview prep tool. Your notes live inside the HTML file itself — copy it anywhere and everything travels with it. No account, no server, no cloud.
✏️
Set the role and company in the header. Switch between Questions to Ask (for you to ask the panel) and Questions to Prepare (for you to answer). Add your own questions, reorder, and take notes on each.
Linking the file — portable saving
  1. Save this HTML file somewhere permanent (e.g. Desktop or Dropbox).
  2. Open it in Chrome or Edge. Click Link file in the yellow banner.
  3. In the file picker that opens, navigate to and select this same HTML file.
  4. Done. Every note you type is now written back into the file within half a second.
Why Chrome/Edge? The File System Access API — which allows the file to rewrite itself — is only fully supported in Chromium browsers. In Safari or Firefox, notes still save to browser storage (localStorage) and you can export to MD, but they won't travel with the file.
During the interview
🎯
Hit Focus in the header. This hides guidance, reorder controls, and delete buttons — leaving just question + notes field + done checkbox. Nothing to accidentally click.
📍
The category pills in the sub-header stay fixed as you scroll and highlight your position. Click any pill to jump to that section instantly.
✅
Tick questions off as you go. The progress counter (e.g. 3 / 14) tracks your current tab.
Guidance badges
Short 1–2 sentences. Clarifying question — listen and note specifics. Listen Open-ended. Let them talk. Note what they emphasise and omit. STAR story Structured 3–4 min answer: Situation, Task, Action, Result. Technical 1–2 min. Specific tools, examples, scale. Avoid vague generalisations. Medium 2–3 min. Deliberate, structured — show your thinking process. Motivation 1–2 min. Be genuine and specific. They can tell the difference. Concise Answer directly. Don't over-explain or pad.

Managing questions
＋
Click + Add question next to any category header, or the blue + button at bottom-right. You can create new categories by typing a new name in the Category field.
⠿
Hover any card to reveal the drag handle on the left. Drag to reorder within a category. Or use the ↑ ↓ buttons on the right.
✕
